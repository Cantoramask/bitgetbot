# Bitget Bot

Install:
1) `python -m venv .venv && source .venv/bin/activate`
2) `pip install -r requirements.txt`
3) copy `.env.example` to `.env` and fill keys
4) `python app.py`

Notes:
• Run in paper mode first (LIVE=false).  
• Logs in `logs/`, JSONL events in `data/journal/`.  
• State in `data/state/`.
